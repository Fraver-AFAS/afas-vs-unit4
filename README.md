# AFAS vs Unit4 presentatie

Deze map bevat een korte, presenteerbare one-page website voor een interne vergelijking tussen AFAS Software en Unit4.

## Bestanden
- `index.html`: inhoud en slidestructuur
- `styles.css`: AFAS-branded visuele stijl
- `script.js`: toetsenbordnavigatie en actieve sectie-indicatie

## Gebruik
1. Open `index.html` in een browser.
2. Gebruik spatie of de pijltjestoetsen om tussen secties te navigeren.
3. Presenteer de pagina fullscreen voor een slide-achtig effect.

## Inhoudsstructuur
1. Introductie en positionering
2. Positionering AFAS versus Unit4
3. Functionele overlap
4. Doorslaggevende verschillen
5. Keuzehulp per organisatietype
6. Eindconclusie waarom AFAS vaak de betere keuze is

## Opmerking
De toon is neutraal vergelijkend, met een duidelijke AFAS-conclusie op het einde. Dat past beter bij een interne presentatie dan een puur commerciële pitch.
